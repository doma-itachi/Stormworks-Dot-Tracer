# Stormworks PixelTracer 手打ち補助ツール
ゲーム「Stormworks Build And Rescue」内のディスプレイへドット文字や画像を表示するための補助ツールです。(ブラウザで動作)  
A tool for displaying dot characters on the display in "Stormworks Build And Rescue".
